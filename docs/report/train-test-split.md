# Train/Test Splitting

The data set is split into train/validation/test sets using stratified splitting method which the same class distribution among all the three sets. This is very useful for the imbalanced distribution like the current case.

| Class       | Count  | Percentage |
|-------------|--------|------------|
| Train       | 300    | 70         |
| Validation  | 65     | 15         |
| Test        | 65     | 15         |
